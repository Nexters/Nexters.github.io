## 작성방법
1. _posts 폴더에 년도-월-날짜-제목.md 파일을 만든다. ex) 2014-08-14-내용.md
2. jekyll serve --watch 를 입력해서 서버를 킨다.
3. 내용을 계속 작성
4. 완료후에 rake site:publish 를 해준다.

## 설치
- gem install kekyll --pre
- bundle install

## Run
- jekyll serve --watch

## 올리기
- rake site:generates
- rake site:publish
